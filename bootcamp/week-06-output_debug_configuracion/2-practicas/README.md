# Practicas - Semana 6

Esta semana las practicas se enfocan en medir, depurar y configurar curl de forma
sistematica. Cada ejercicio construye sobre el anterior.

## Orden recomendado

1. **01-ejercicio-write-out**: extraccion de metricas basicas
2. **02-ejercicio-debug**: depuracion con verbose y trace
3. **03-ejercicio-curlrc**: configuracion persistente
4. **04-ejercicio-metricas**: script de benchmark completo

## APIs que usaremos

- `https://jsonplaceholder.typicode.com` - API REST publica, sin autenticacion
- `https://httpbin.org` - Herramienta de inspeccion de requests HTTP

## Como entregar

Para cada ejercicio, guarda en el directorio correspondiente:
- Los comandos que ejecutaste (en un archivo `solucion.sh` o similiar)
- El output capturado (en un archivo `output.txt`)
- Respuestas a las preguntas planteadas (en un archivo `respuestas.md`)
