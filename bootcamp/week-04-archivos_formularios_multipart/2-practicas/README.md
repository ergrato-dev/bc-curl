# Practicas - Semana 4

Cuatro ejercicios progresivos sobre descarga y upload de archivos. Completal en orden.

## Ejercicios

| # | Carpeta | Tiempo | Objetivo |
|---|---------|--------|----------|
| 01 | [01-ejercicio-descargas](01-ejercicio-descargas/) | 45 min | Descargar archivos con -o, -O, --progress-bar, reanudar |
| 02 | [02-ejercicio-form-post](02-ejercicio-form-post/) | 45 min | Enviar formularios URL-encoded con caracteres especiales |
| 03 | [03-ejercicio-upload](03-ejercicio-upload/) | 60 min | Subir archivos con multipart usando -F |
| 04 | [04-ejercicio-combinado](04-ejercicio-combinado/) | 90 min | Request con texto + archivo, simular perfil de usuario |

## Como entregar

Crea un archivo `respuestas.md` en cada carpeta de ejercicio con:
- Los comandos que usaste (exactos, copiados de tu terminal)
- El output obtenido o los primeros 30 líneas si es largo
- Tu respuesta a las preguntas de reflexión

## Requisitos

- curl 7.x instalado
- Acceso a `https://httpbin.org`
- Un directorio de trabajo limpio para los archivos descargados
