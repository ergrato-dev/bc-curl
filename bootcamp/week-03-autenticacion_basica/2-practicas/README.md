# Practicas - Semana 3

Cuatro ejercicios que cubren los principales mecanismos de autenticación. Completal en orden.

## Ejercicios

| # | Carpeta | Tiempo | Objetivo |
|---|---------|--------|----------|
| 01 | [01-ejercicio-basic-auth](01-ejercicio-basic-auth/) | 45 min | Basic Auth con `-u`, observar header, probar 401 |
| 02 | [02-ejercicio-api-key](02-ejercicio-api-key/) | 45 min | API Key en header vs query string |
| 03 | [03-ejercicio-bearer-token](03-ejercicio-bearer-token/) | 60 min | Flujo login → token → request autenticado |
| 04 | [04-ejercicio-variables-entorno](04-ejercicio-variables-entorno/) | 60 min | Script con credenciales en variables de entorno |

## Como entregar

Crea un archivo `respuestas.md` en cada carpeta de ejercicio con:
- Los comandos exactos que usaste
- El output obtenido
- El status code de cada request
- Las respuestas a las preguntas de análisis
