# Practicas - Semana 2

Cuatro ejercicios progresivos. Completal en orden — cada uno construye sobre el anterior.

## Ejercicios

| # | Carpeta | Tiempo | Objetivo |
|---|---------|--------|----------|
| 01 | [01-ejercicio-post](01-ejercicio-post/) | 45 min | Crear recursos con POST, verificar 201 |
| 02 | [02-ejercicio-crud](02-ejercicio-crud/) | 90 min | Ciclo CRUD completo sobre el mismo recurso |
| 03 | [03-ejercicio-headers](03-ejercicio-headers/) | 45 min | Experimentar con headers, observar en httpbin |
| 04 | [04-ejercicio-json-archivo](04-ejercicio-json-archivo/) | 60 min | Trabajar con JSON desde archivos externos |

## Como entregar

Crea un archivo `respuestas.md` en cada carpeta de ejercicio con:
- Los comandos exactos que usaste
- El output obtenido (primeras 30 líneas si es largo)
- El status code de cada request
- Tu interpretación de la respuesta
